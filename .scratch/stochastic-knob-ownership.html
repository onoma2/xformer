Stochastic knobs — behavior, ownership, layer, RNG, law

Five orthogonal axes per knob. Behavior (section) = how the value reaches playback at runtime. Ownership = where the data lives in the model. Layer = whether the knob is structural/config (model) or a runtime decision parameter (engine). RNG = what randomness source runs under the knob. Law = mathematical shape of how the value translates to effect.

REALTIME engine reads knob directly at trigger time — no intermediate storage
LOOP Loop: cache picks per step from knob (scrubable). Live: engine rolls per step from knob.
LIVE-ONLY Live: engine rolls per step. Loop: cache reads steps array verbatim; knob inert until next RENEW / Mutate.
CYCLE-END fires at cycle boundary — may rotate seed and rewrite events
PATTERN SHAPE bounds the playback window; no per-step content effect
SEQ StochasticSequence — per-pattern. Each pattern has its own copy. Changes when pattern index changes during playback (changePattern repoints _sequence).
TRK StochasticTrack — per-track global. All patterns of this track share the value. Stays the same across pattern switches.
Note: saving the project persists both SEQ and TRK fields — save survival is universal. Ownership only governs pattern-switch behavior.
MODEL structural / config — engine respects but doesn't probabilistically decide with (scale, tempo, window, track offsets)
ENGINE runtime decision parameter — engine uses for picks, rolls, scaling, filtering
KEYED cache walk uses keyed_rng::cellSeed(seed, stepIndex) — same seed + same knobs = same output, scrubable
ENGINE engine's _rng state — advances per call, non-reproducible across rebuilds
NONE no random roll involved — knob value used directly (scalar multiplier, threshold, integer offset)
SCALAR continuous numeric used directly (multiplier, tau, %)
BIPOLAR signed continuous — sign carries meaning beyond magnitude
BERNOULLI yes/no probability roll per step or cycle
LUT CENTER picks a discrete slot from a finite table (often paired with a kernel width)
KERNEL WIDTH width of a distribution around a center value
ENUM selects from named options
POISSON Poisson lambda — CDF rises with each cycle until threshold
INTEGER discrete count / offset / position
THRESHOLD value compared against ranked input — steps pass or get filtered

Feel sequence.feel() SEQ Engine None Scalar Per-trigger Q16.16 scale on step duration toward 3/4 or 5/4 meter. Detent at 45..55. Fixed
Repeat sequence.repeatProb() SEQ Engine Engine Bernoulli Per-trigger Bernoulli — replays _lastEvent verbatim instead of advancing. Fixed
MaskR sequence.maskRhythm() SEQ Engine None Threshold Per-trigger duration-rank threshold — steps whose stored rank exceeds the threshold become rests. Paired with TiltR: MaskR sets cutoff %, TiltR picks which end of the duration axis to cut. LoopR-only: LiveR events bypass generateMaskRanks and store rank=0, so the engine gates the whole block on rhythmMode==Loop. Knob is inert in LiveR. Fixed
TiltR sequence.tiltRhythm() SEQ Engine None Scalar (centered) Duration filter direction + strength, knob 0..100 centered at 50. TiltR=50 = pure salt cut (random thinning); TiltR<50 = low-pass (long notes survive), TiltR>50 = high-pass (short notes survive). Distance from 50 sets the rank-vs-salt blend (|knob-50|×2 = magnitude). Pure trigger-time read. LoopR-only — same gate as MaskR; inert in LiveR. Fixed
MaskM sequence.maskMelody() SEQ Engine None Threshold Per-trigger pitch-centrality threshold — steps whose played degree scores below the threshold become rests. Centrality comes from stochasticPitchCentrality() (root/half/thirds/quarters with triangular kernels), normalized against the scale's root-anchor weight. LoopM-only: LiveM pitches are rolled fresh each trigger so the filter has no stable material to thin against; the engine gates the whole block on melodyMode==Loop. Knob is inert in LiveM. Default 100 = bypass. Pitch-centrality is no longer a factor in the generator's pick — this knob owns it entirely. Fixed
TiltM sequence.tiltMelody() SEQ Engine None Scalar (unipolar) Inversion magnitude on the MaskM filter. TiltM=0 keeps the highest-centrality steps (tonal — root, fifth, thirds survive); TiltM=100 inverts to keep the lowest-centrality steps ("everything but the root"); intermediate values progressively blend the two sets. Pure trigger-time read. LoopM-only — same gate as MaskM; inert in LiveM. Fixed
Divisor sequence.divisor() SEQ Model None LUT center Sequence base clock divisor — picks ticks-per-step from a small set of musical subdivisions. Fixed
Clock Multiplier sequence.clockMultiplier() SEQ Model None Scalar Multiplier on step duration (50..200 %). Combined with divisor. Fixed
Reset Measure sequence.resetMeasure() SEQ Model None Integer Hard sync — sequence restart every N measures (0 = off). Fixed
Slide Time track.slideTime() TRK Model None Scalar Track-level slide tau (0..100). Engine uses uniformly across all events with slide set. Fixed
Scale track.scale() TRK Model None Enum Selected scale for pitch generation + voltage conversion. Fixed
Root Note track.rootNote() TRK Model None Enum Root note offset for chromatic scales. Fixed
Octave track.octave() TRK Model None Integer Signed track-wide octave offset applied at CV computation. Fixed
Transpose track.transpose() TRK Model None Integer Signed track-wide semitone offset. Fixed
Fill Mode track.fillMode() TRK Model None Enum Behavior under fill mute. Fixed

Note Duration sequence.noteDuration() Rhythm SEQ Engine Keyed LUT center Home center of the duration LUT (x8 .. x1/2 of divisor). The picker is a three-zone Variation blend: at V=0 only the home bell contributes → strict NoteDuration lock. As Variation grows past zero, ticket and explore zones blend in. See Variation row. Fixed
Variation sequence.variation() Rhythm SEQ Engine Keyed Kernel width Owns which duration-pick zone dominates per cell. Three overlapping triangles: home (peaks V=0), ticket (peaks V=50), explore (peaks V=100). At V=0 NoteDuration locks; at V=50 Duration Tickets drive; at V=100 explore wins (floor + opposite-side bell). Duration tickets are three-state (-1 exclude, 0 flat=1, 1..100 emphasis), same contract as pitch tickets. Fixed
Range sequence.range() Melody SEQ Engine Keyed Integer Bipolar 0..100 centered at 50. Range=50 → single-octave field (Steps 3-5 of the pitch law collapse to no-ops). Range>50 → field expands up to 4 octaves at knob=100. Range<50 → single octave + per-slot octave-displacement chance, stronger toward knob=0. Bias/Spread/Contour all operate on the candidate set built here. Fixed
Burst sequence.burst() Rhythm SEQ Engine Keyed Bernoulli Per-step cluster-start probability. On hit: cluster timing depends on BurstHold's Fit/Over axis (see BurstHold row). Floor: cluster cells must each be ≥ kMinClusterCellTicks (12 ticks) so the gate clamp leaves a discrete retrigger gap. Below that floor the cluster doesn't fire — cell plays as ordinary. Fixed
Burst Count sequence.burstCount() Rhythm SEQ Engine Keyed LUT center Center of triangular pick over BurstCount LUT {2,3,4,5,6,7,8}. Returns TOTAL cluster cells (anchor + tails). Stored as burstTails = total - 1 in the 3-bit event field. In Fit mode, if prev_dur can't house the requested count at the 12-tick floor, count auto-reduces to whatever fits. Fixed
Burst Rate sequence.burstRate() Rhythm SEQ Engine Keyed LUT center Two interpretations depending on BurstHold timing axis. Fit modes: log-symmetric curve factor r ∈ [0.4..2.5] (knob=0 → r=0.4 strong accel, knob=50 → r=1.0 uniform, knob=100 → r=2.5 strong decel). Cluster cell durations follow r^k normalized to sum = prev_dur. Overflow modes: triangular pick over spacing-denominator LUT {2,3,4,5,6}; cluster cells uniform at prev_dur / denom. Fixed
BurstHold (4-state) sequence.burstHold() Both SEQ Engine Keyed Enum Single 4-state enum collapsing two orthogonal axes. Pitch axis: Hold = cluster cells share anchor pitch; Roll = each cluster cell re-rolls own pitch via melodySeed-keyed RNG. Timing axis: Fit = cluster cells distributed across one prev_dur via BurstRate curve (trap-style flams); Over = cluster cells uniform at prev_dur/denom, can overflow into next slot budget (today's classic burst). States: HoldOver (default — uniform shared-pitch), HoldFit, RollOver, RollFit. Fixed
Complexity sequence.complexity() Melody SEQ Engine Keyed Kernel width 5-step pitch law — Step 2 (class repeat check). Probability of REJECTING a class-repeat: 0 = repeats kept (stepwise feel), 100 = forced movement (every same-class roll re-rolls Step 1). Bounded retries (4) so heavy single-ticket emphasis never hangs the picker. Does NOT shape candidate pool — that's Range/tickets. Fixed
Contour sequence.contour() Melody SEQ Engine Keyed Bipolar 5-step pitch law — Step 5 (direction nudge). Applied AFTER Step 4 region roll. If chosen pitch's direction (vs lastPitch) mismatches Contour sign, roll swap = rng < |Contour|; on hit, replace with the nearest pitch in the candidate set that goes the desired direction. Bounded retries (2). Octave-blind — consumes the sorted candidate set as a flat list. Fixed
Bias sequence.marblesBias() Melody SEQ Engine Keyed LUT center 5-step pitch law — Step 4 region roll, beta-distribution shape center. 0 → low end of candidate set, 50 → center, 100 → high end. Combined with Spread to drive the beta sample; the result indexes into the sorted candidate set (octave-blind). Fixed
Spread sequence.marblesSpread() Melody SEQ Engine Keyed Kernel width 5-step pitch law — Step 4 region roll, beta-distribution concentration. Spread<50 collapses toward Bias (Spread=0 = pure Bias). Spread=50 = cloud around Bias. Spread>50 Bernoulli-pulls toward edges with Bias picking the edge. Linear weighting replaced by continuous beta-shape. Fixed
Gate Length sequence.gateLength() Rhythm SEQ Engine Keyed Kernel width Width of triangular distribution around 50% center. Cache picks per step via two keyed RNG pulls — knob is a deterministic playable control over the per-slot curve; same value recalls the same set. Fixed
Slide sequence.slide() Melody SEQ Engine Keyed Bernoulli Per-step Bernoulli for glide-into-this-note. When set, the engine ramps CV toward the new pitch over the Slide Time (track-owned). Cluster tails inherit the anchor's slide decision. Fixed
Legato sequence.legatoProb() Rhythm SEQ Engine Keyed Bernoulli Per-step Bernoulli for tie-into-next. When set, the gate stays at 100% and the gate-off is suppressed so the step joins seamlessly into the next slot. Cluster tails inherit the anchor's legato decision. Fixed

Rest sequence.rest() SEQ Engine Keyed (Loop) / Engine (Live) Bernoulli Per-cell Bernoulli for silence. Loop mode: rolled per-cell at cache time from a salt-separated RNG cellSeed(rhythmSeed ^ 0x5E57DEAD, K) — independent dice from the duration roll. Moving the Rest knob reshapes the audible mask without rerolling durations; moving NoteDuration/Variation/Tickets doesn't reshuffle rests. Live mode: engine writes ev.rest() per trigger using _rng. Cluster tails inherit anchor's rest decision. Fixed

Cycle-end — fires at cycle boundary, may rotate seed or rewrite steps array

UI labelCode fieldOwnerLayerRNGLawBehaviorState
Patience R sequence.patienceRhythm() SEQ Engine Engine Poisson Poisson CDF with cycle-count buildup. Knob → λ on a log2 curve: every ~14 knob units doubles the expected wait (knob 0 → λ=1, knob 14 → λ=2, knob 28 → λ=4, ..., knob 99 → λ=128). Knob 100 = off sentinel (never regenerate). On fire: rotates rhythmSeed, regenerates the entire rhythm steps array. Fixed
Patience M sequence.patienceMelody() SEQ Engine Engine Poisson Same as Patience R, melody domain. Fixed
Mutate sequence.mutate() SEQ Engine Engine Bernoulli Unipolar 0..100. Per-cycle probability of destructively re-rolling one event inside [first..last]. Domain (rhythm or melody) picked at random when both are Loop+valid. Rhythm mutation re-runs generateMaskRanks globally so MaskR stays correct over the new content. Melody mutation routes through the 5-step pitch law (generateDegree with fresh PitchGenState). Permute branch dropped — Mutate is destructive-only now. Fixed
Jump sequence.jump() SEQ Engine Engine Bernoulli Per-cycle Bernoulli. On fire: _jumpRegister walks ±1 octave, bounded ±2. Applied as octave offset at trigger time. Fixed
Sleep sequence.sleep() SEQ Engine None Integer Per-cycle: skip N cycles before next playback (N = knob × 4 / 10). Fixed

Pattern shape — playback window bounds (SEQ-owned, no per-step content effect)

UI labelCode fieldOwnerLayerRNGLawBehaviorState
First sequence.first() SEQ Model None Integer Playback start slot. Cells are slot-keyed, so moving First plays the same slots in their same shape from a different starting point — no cache rebuild, no per-step RNG shift. Fixed
Size sequence.size() SEQ Model None Integer Total pattern slot count (2..64). Size also acts as the last-slot bound — playback runs first .. size-1. Growing Size extends the cache walk to cover newly-active slots (triggers a refresh); shrinking just truncates the active range. Fixed
Rotate sequence.rotate() SEQ Engine None Integer Signed read-offset added to the in-window playback position at trigger time. Polyrhythmic phasing against other tracks — steps stay slot-keyed; Rotate just changes which slot fires first within the active window. Fixed

Last is stubbed in the model — sequence.last() resolves to size - 1, setLast() is a no-op. The UI no longer exposes it.
