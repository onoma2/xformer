Teletype ops — upstream vs TT2 native table

Branch feat/tt2-v2-native. Scoped to the computational/musical core (the families TT2 actually targets). Upstream hardware-module ops — ansible, telex, grid, MIDI, just friends, WW — are out of scope by design and not listed.

Control-flow language is complete and not in this table — IF / ELIF / ELSE, PROB, EVERY, SKIP, L (loop), the DEL family (DEL / .X / .R / .G / .B), and the ms metro are handled as pre-op modifiers in lowering + the runner, not as native table ops. Only their op-side hooks (BREAK, BRK, KILL, SCRIPT, DEL.CLR, M, M.ACT) live in the table.
