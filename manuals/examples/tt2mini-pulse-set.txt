NAME MINI PULSE
MINI PULSE - a 4-scene TT2-Mini set (one evolving performance: intro -> build
-> peak -> outro). TT2-Mini has scripts 1, 2 and M only (no Init), 2 trigger
inputs, 4 patterns. Each scene below is a complete Mini program; the four
together fill the track's four scene slots.
Patch (all scenes): TR 1 -> kick, TR 2 -> hat, CV 1 -> bass, CV 2 -> pluck.
