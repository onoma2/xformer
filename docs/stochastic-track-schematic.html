Stochastic Track Control Map

STOCH

The Performance Page

Generator: Complexity and Contour define the "melody shape." Low complexity favors repetition and small intervals.

Rhythm: Rate maps 0-100 to absolute notes (Bar to 1/128). Variation introduces bipolar shifts to slower (+) or faster (-) durations.

Bursts: Probabilistic child events (ratchets) inside parent durations. Count (1-4) and Rate (timing compression) define the trill.

Dice: Uses the pattern buffer window. Rotate provides a non-destructive phase shift of the loop.

STOCH CFG

The Config Page

Infrastructure: Standard Performer timing. Divisor and Reset Measure allow poly-metric relationships.

Pitch Context: Root and Scale determine how generated degrees are quantized to volts. Octave/Transpose apply static offsets.

CV Engine: Slide Time controls the slew speed for slide events. CV Update determines if CV moves only on gates or always.
