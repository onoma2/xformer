What Chooses The Pitch?

A musician-facing map of the Stochastic track pitch path. It shows which controls shape the note before it reaches the oscillator.

tonal home note palette melodic motion performance offsets change over time

Read it like a band rehearsal: first choose the key, then choose the allowed note palette, then decide how adventurous the melody is, then shape the odds, then move the whole phrase up or down.

Stable melody Use Loop, narrow Range, gentle Complexity, and clear Pitch Tickets.
Searching melody Use Live, wider Range, more Spread, and more Complexity.
Phrase movement Use Contour, Octave, Transpose, and occasional Jump.
