Pitch Of One Stochastic Note

The note you hear is not white noise. One pitch is shaped by a remembered melody, a tonal home, a note palette, melodic pull, and final height controls.

Stochastic Track

Remembered melody

Melody, Repeat, NewM, Patience M, and Mutate decide whether this note comes from the current phrase or a changed phrase.

Available notes

Scale, Root Note, Pitch Tickets, Range, Low Degree, High Degree, and Steps set the note palette.

Musical pull

Complexity, Contour, Bias, and Spread shape which available note feels more likely.

Played height

Octave, Transpose, and Jump move the chosen note to the pitch CV the oscillator receives.
