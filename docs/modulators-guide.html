1 · What modulators are

XFORMER has 8 modulators (M1–M8), separate from your sequencer tracks. Each is a small signal generator — an LFO, a random source, an envelope, a chaos attractor — whose output you route to CV jacks and/or MIDI to move things around. They run continuously, even when the transport is stopped (Free-rate modulators keep going on wall-clock time).

Open the modulator page, pick a modulator with the track-select keys (1–8 select M1–M8), and you get a live scope of its output on the left and its parameters on the right.

Additive by nature. A modulator assigned to a CV output is summed into whatever that jack already carries — a bipolar offset, not a replacement. Value 64 (0 V) adds nothing; above nudges the output up, below nudges it down. So modulators layer on top of the routing like a stackable attenuverter, and the resting centre/floor matters: park it at 64 and the destination is untouched until the modulator moves.

3 · From value to voltage

Internally a modulator outputs a number 0–127. At a CV jack that maps bipolar around centre:

Value064127
Volts−5 V0 V (centre)+5 V

So 64 is the resting centre / 0 V. A bipolar LFO swings around 64; a unipolar envelope rests at its floor (64 by default = 0 V) and rises toward 127. The same value also feeds MIDI and the CV-route Mod sources, so what you see in the scope is what every destination gets.

Random has two flavours: Clocked steps to a new smoothed value on its own rate; Triggered is a sample-&-hold that grabs a new value on each gate (the third cell becomes G.SRC, the gate source). Chaos (Lorenz / Latoocarfian) are strange attractors — RATE sets how fast you travel the attractor, P1/P2 reshape it.

5 · Rate: Free vs Tempo

RATE has two domains; re-press the RATE key to flip between them:

FREE
Wall-clock Hz, 0.01–16 Hz. Tempo-independent — runs even when the transport is stopped. Reads like 2.00Hz.
TEMPO
A clock division, locked to the project tempo. Freezes when the transport stops — that's what sync means.

6 · Gate modes & sources

A modulator can react to a gate. The gate mode (footer RUN on the destinations page) sets how:

ModeBehaviour
FREEFree-running; the gate is ignored.
TRIGReset the phase to its offset on every gate rising edge (hard sync).
HOLDAdvance only while the gate is high; reset on rising.

The gate source (footer GATE) can be any of:

  • Track gates — GateOut1–8
  • CV inputs — CvIn1–4 (crosses at ~1 V)
  • Routing buses — BusCv1–4
  • Other modulators — Mod1–8 (a modulator can't gate from itself)

Continuous sources are thresholded at their midpoint with a little hysteresis, so an LFO or CV can act as a gate.

7 · Floor & Invert

The output passes through a per-modulator floor + invert just before it becomes voltage, so every destination stays consistent.

  • FLOOR = the OFFSET value, read as the rest voltage. Default 0 V; an envelope rests here and rises. On ADSR it's a cell on page 2 (shown in volts).
  • INVERT = flips the output around its range. On an envelope it rests high and ducks down — a sidechain shape. (ADSR page 2 toggle.)

ADSR page 2 — AMPLITUDE, plus FLOOR (rest voltage) and INVERT.

8 · Destinations (routing)

Press → past the last parameter page to reach destinations. A modulator can drive many outputs at once — a membership grid of CV jacks (1–8) and MIDI outs (1–16); filled = assigned. The encoder moves the cursor, push adds, CLEAR (F3) removes the cursored one.

Destinations — CV/MIDI membership grid + the cursored output's EVENT/CC, and RUN/GATE for the modulator.

Per MIDI out, the EVENT (F4) chooses what the modulator sends — CC (with a CC NUM), Bend (pitch bend), or Aftertouch (channel pressure). Footer keys also carry the gate RUN mode and GATE source from §6. Changes commit through the context menu (CANCEL / COMMIT on F4/F5).

9 · JustF — the rate-link mode advanced

A transient mode (toggle Shift+RATE) borrowed from Instruo ochd / Just Friends: it links all eight modulators to M1 through a single INTONE spread. M1 is the master; M2–M8 fan out to harmonic (CW) or subharmonic (CCW) multiples.

  • On LFO/Chaos followers it spreads their Free rate (M8 up to 8× M1, B-clamped so the fastest stays ≤ 16 Hz).
  • On ADSR followers it spreads their envelope times instead — each follower runs M1's envelope scaled by its index ratio (harmonic = snappier down the bank).
JustF links speeds, not triggers: each ADSR follower still fires on its own gate source. To get one trigger firing the whole related-speed bank, point all the followers at the same gate source.

JustF — M2's RATE cell becomes INTONE; M3–M8 show their derived rate.

JustF is a temporary link, not a saved mode: turning it off bakes the derived rates/times into each modulator and leaves. It's mutually exclusive with Geode.

10 · Geode — the rhythmic engine advanced

Geode turns the whole bank into a 6-voice Just Friends: each voice fires bursts of envelopes on a gate, and the six are tuned together by INTONE. Toggle with Shift+SHAPE (mutually exclusive with JustF). Unlike JustF, Geode is a real saved mode — it persists with the project.

How the bank becomes a Geode

Geode conscripts the whole bank: no independent modulators while it's on. Turn it off and M3–M8 snap back to their normal shapes — nothing destroyed. On entry it parks M2 to depth 0 so RUN starts neutral (0).

What a "note" is in Geode

One gate fires a burst, defined per voice by two numbers:

  • DIVS — how finely the measure is sliced (4 = quarters, 15 = fifteen/bar). Higher = faster retriggers.
  • REPEATS — envelope events in the burst. 0 = one hit, 4 = five, −1 (INF) = forever.

RATE vs TIME. M1's RATE (page 1) is the clock — how fast the rhythm grid cycles (your tempo). TIME (page 2) is the envelope length of each hit. Two different axes; in original Just Friends the rhythm comes from a clock/TICK and TIME is the duration knob — same split.

RUN & the three MODES

RUN comes from M2's output (bipolar; 0 at centre = neutral). It reshapes how a burst's amplitudes evolve, and what it does depends on MODE:

MODEBurst behaves like…RUN (from M2) controls…
TRANS accentEvery repeat full velocity — steady stream.Accent cycles (emphasise every n-th hit).
SUST gravityRepeats decay to silence — bouncing-ball / tom roll.Decay rate: + accelerates, − slows.
CYCLE tremoloContinuous amplitude cycling — tremolo.Cycle rate; negative adds pseudo-random feel.
At run 0 each mode gives its clean baseline (full velocity / linear decay / one cycle). To hold run neutral, M2 sits at centre — depth 0 does exactly that (and entering Geode parks it there). A depth-0 + offset M2 gives a fixed run bias with no movement.

The voice page

A voice (M3–M8) — live scope + DIVS/REPEAT. Gate & output live on its destinations page.

Step-by-step: your first Geode patch

  1. Select M1, press Shift+SHAPE. Header reads GEODE.
  2. On M1 page 1 set RATE — the clock. ~1 Hz is a relaxed start.
  3. → to M1 page 2: TIME ~200 ms, INTONE noon, RAMP 0.5, CURVE LIN, MODE TRANS.
  4. Select M3: DIVS 4, REPEAT INF.
  5. → to M3's destinations: pick a gate source (a track gate) and a CV/MIDI out.
  6. Give that track a gate and press play — watch M3's scope fire.
  7. Repeat for M4–M8 with different DIVS, then turn INTONE up on M1.

What to expect

  • A default voice rests at 0 V (centre) and rises — on the scope it uses the upper half (unipolar, not a fault). Drop its FLOOR for a full-scope swing.
  • Gate density matters: a densely-gated track re-fires the burst before the envelope completes (looks frantic/small). Sparse gates → clean full bursts.
  • High DIVS + long TIME = overlapping "machine-gun" hits. Keep DIVS modest relative to TIME.

Five setups to try

1 · Percussive multi-voice TRANS

MODE TRANS · TIME ~150ms · RAMP 0.1 · INTONE noon · DIVS 4/3/6/8/2/5 · REPEATS 0 — program rhythms on tracks 3–8.

Six lanes, each on its own DIVS — full-height plucks (TRANS), one per gate.

2 · Bouncing tom rolls SUST

MODE SUST · TIME ~250ms · REPEATS 8 · DIVS 8 · M2 (RUN) positive — RUN+ accelerates the decay.

One gate, REPEATS 8 = nine evenly-spaced hits that decay to silence.

3 · Tremolo bed CYCLE

MODE CYCLE · REPEATS INF · DIVS 1 · RAMP 0.5 · M2 (RUN) sets rate — negative RUN wobbles.

A continuous train (REPEATS INF) whose peak amplitude rides a slow LFO — tremolo.

4 · Polymetric ostinato TRANS

DIVS 3/5/7 · REPEATS INF · one gate each · TIME short — phases realign only every few bars.

Prime divisions drift against each other; the full pattern repeats only every 3×5×7 bars.

5 · INTONE strum harmonic

all voices one gate · REPEATS 0 · same DIVS · sweep INTONE — high voices snap, low ones linger.

All six fired together; INTONE sets six related envelope lengths from a single trigger.

Nothing firing? A voice only sounds when its gate source fires. Check the voice's destinations page, make sure that track has a gate, and that the transport is playing (or M1 is on a Free clock). REPEATS = INF means a single gate keeps it going.
