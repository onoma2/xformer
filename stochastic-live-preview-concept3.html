Concept 3: Mycelium Path

Inspiration: flora.lua (L-system branching) + luck.lua (node ring topology) + less-concepts (cellular growth).
Poetic framing "The sequencer is a living fungus. Each note is a node that sends a hypha toward the next. Pitch determines branch direction. Rests are dead ends. The network grows from right to left, newest growth tip pulsing with life."

How it works

  • Nodes: each event is a branching point (a small filled circle) in a mycelial network.
  • Hyphae: thin filaments connect consecutive non-rest events. The filament curves organically — not straight lines — using a slight sine wobble so the network feels alive.
  • Pitch = branch direction: higher pitch sends the hypha upward; lower pitch sends it downward. The angle is gentle (not jagged) so the network flows.
  • Newest node: the rightmost growth tip pulses with a bright halo and a bioluminescent glow.
  • Rests: terminate the current hypha with a dark "dead end" stub. The next note starts a new branch from a slightly offset position, creating forking.
  • Burst children: spore clusters — small radiating dots around the parent node, like fungal sporangia.
  • Slide: thickens the hypha into a luminous "mycelial cord."
  • Legato: keeps the hypha at high brightness between nodes, as if the tissue is continuously alive.
  • Range: opens the vertical spread of the entire colony — more range = taller, more arborescent growth.
  • Contour: tilts the growth direction globally, making the colony lean uphill or downhill.

Why this works

  • The organic curve language (quadratic bezier hyphae) is softer and more poetic than straight line segments.
  • Rests as dead ends create natural branching — the fungus forks around silence.
  • Burst reads as biological reproduction (spores), not technical scheduling.
  • The growth direction from right-to-left matches reading direction and time flow intuition.
  • Very low pixel count — OLED-friendly — yet visually rich due to connection topology.
