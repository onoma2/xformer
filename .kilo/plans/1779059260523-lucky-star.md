# Plan: Unified Shape Encoder (Lorenz → LFO Morph → Random → Latoocarfian → ADSR)

## Goal
Redesign `editShape()` so the encoder ticks through a unified virtual index:
- **0** = ChaosLorenz
- **1–100** = LFO morph (Sine→Square, interpolated waveform)
- **101** = Random
- **102** = ChaosLatoocarfian
- **103** = ADSR

The 5 old LFO enum values collapse into a single LFO mode driven by `_shapeMorph` (0–99 internal, 100 steps). Grid shows interpolated shape names ("Sine", "Tri", "Saw+", "Saw-", "Sqr").

## Verified Topology

- **State ownership**: `_shapeMorph` lives in `Modulator` model (1 byte/modulator). Existing `_shape` enum kept stable for serialization compat.
- **Feedback**: interpolated name shown in grid cell 0; waveform cache draws blended shape in real time.
- **Blast radius**: `Modulator.h` (enum semantics, `editShape` rewrite), `WaveformGenerator.h` (new `generateLFO`), `ModulatorEngine.h` (LFO branch uses morph), `ModulatorPage.cpp` (cache invalidation + display).
- **Timing**: `editShape` is synchronous UI→model; engine reads morph on next `tick()` — no race.

## Model Changes (`model/Modulator.h`)

Keep existing `Shape` enum **unchanged** (Sine=0…Square=4…ChaosLatoocarfian=8) so serialization format stays byte-compatible.

Add:
```cpp
uint8_t _shapeMorph = 0;  // 0–99, only meaningful when isLFOShape(_shape)
```

Add helpers:
```cpp
static bool isLFOShape(Shape shape) {
    return shape == Shape::Sine || shape == Shape::Triangle ||
           shape == Shape::SawUp || shape == Shape::SawDown || shape == Shape::Square;
}
int shapeMorph() const { return _shapeMorph; }
void setShapeMorph(int morph) { _shapeMorph = clamp(morph, 0, 99); }
```

Rewrite `editShape()` with virtual index mapping:
```cpp
void editShape(int value, bool shift) {
    // Virtual index: 0=Lorenz, 1-100=LFO, 101=Random, 102=Latoocarfian, 103=ADSR
    int current;
    if (shape() == Shape::ChaosLorenz) current = 0;
    else if (isLFOShape(shape())) current = 1 + shapeMorph();
    else if (shape() == Shape::Random) current = 101;
    else if (shape() == Shape::ChaosLatoocarfian) current = 102;
    else current = 103;  // ADSR

    int step = (current >= 1 && current <= 100) ? (shift ? 1 : 5) : 1;
    int next = clamp(current + value * step, 0, 103);

    if (next == 0) setShape(Shape::ChaosLorenz);
    else if (next <= 100) { setShape(Shape::Sine); setShapeMorph(next - 1); }
    else if (next == 101) setShape(Shape::Random);
    else if (next == 102) setShape(Shape::ChaosLatoocarfian);
    else setShape(Shape::ADSR);
}
```

Update `shapeName()` to return interpolated names for LFO mode:
```cpp
static const char* lfoShapeName(int morph) {
    if (morph < 13) return "Sine";
    if (morph < 38) return "Tri";
    if (morph < 63) return "Saw+";
    if (morph < 88) return "Saw-";
    return "Sqr";
}
```

Serialization (`write`/`read`): append `_shapeMorph` at the end. No version bump (dev policy). Old project files on this dev branch may not load correctly — accepted dev-stage cost, no migration required.

## Engine Changes

### `engine/WaveformGenerator.h`

Add `generateLFO(int morph, uint16_t phase)`:
```cpp
static int generateLFO(int morph, uint16_t phase) {
    int segment = morph / 25;        // 0..3
    int blend = morph % 25;          // 0..24
    int left, right;
    switch (segment) {
    case 0: left = generateSine(phase); right = generateTriangle(phase); break;
    case 1: left = generateTriangle(phase); right = generateSawUp(phase); break;
    case 2: left = generateSawUp(phase); right = generateSawDown(phase); break;
    default:left = generateSawDown(phase); right = generateSquare(phase); break;
    }
    return left + ((right - left) * blend) / 25;
}
```

### `engine/ModulatorEngine.h`

In the LFO branch (line ~297), replace:
```cpp
int value = WaveformGenerator::generate(modulator.shape(), phase & 0xFFFF);
```
with:
```cpp
int value;
if (Modulator::isLFOShape(modulator.shape())) {
    value = WaveformGenerator::generateLFO(modulator.shapeMorph(), phase & 0xFFFF);
} else {
    value = WaveformGenerator::generate(modulator.shape(), phase & 0xFFFF);
}
```

All other shape comparisons (`Random`, `ADSR`, `ChaosLorenz`, `ChaosLatoocarfian`) remain valid because the enum is unchanged.

## UI Changes (`ui/pages/ModulatorPage.cpp` + `.h`)

1. **Grid display (cell 0)**: if `isLFOShape()`, show `Modulator::lfoShapeName(modulator.shapeMorph())`. Otherwise show `shapeName()`.
2. **Waveform cache invalidation**: add `int _lastShapeMorph = 0;` to `ModulatorPage.h`; include `modulator.shapeMorph() != _lastShapeMorph` in the dirty check.
3. **`updateWaveformCache()`**: for LFO shapes, call `WaveformGenerator::generateLFO(modulator.shapeMorph(), phase)` instead of `generate()`.
4. **`_lastShapeMorph` update**: set it alongside `_lastShape` in `updateWaveformCache()`.

## RAM Impact

+1 byte per modulator × 8 = **+8 bytes `.bss`** — negligible. No engine container or model gate changes.

## Deferred / Flagged

- **Serialization backward compat**: Appending `_shapeMorph` without a `ProjectVersion` guard means old project files will misalign. Must add `Version35` and guard the read before merging to main.
- **Old→new shape migration**: When version guard is added, old files with `Shape::Triangle` (1) etc. should be mapped to `Shape::Sine` + `shapeMorph=25` on load.
